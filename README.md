# Birthday Engineering Cards

Легкий статический сайт с персональными поздравительными открытками для
инженеров команды. Проект не требует сборки и внешних зависимостей.

## Структура

```text
.
├── index.html
├── assets/
│   ├── css/style.css
│   └── js/main.js
├── cards/template/
├── opsmon/
└── pablo/
```

Каждая открытка находится в отдельной директории с собственным `index.html` и
`task.md`. Благодаря этому URL вида `/opsmon/` и `/pablo/` работают на GitHub
Pages без клиентского роутера.

## Локальный запуск

Из корня репозитория:

```bash
python3 -m http.server 8000
```

Открыть `http://localhost:8000/`.

## Как добавить открытку

1. Скопировать `cards/template` в новую директорию, например `ivan`.
2. Обновить `ivan/index.html`: имя, дату, текст и инженерный status-блок.
3. Обновить `ivan/task.md`.
4. Не добавлять ссылку на главную страницу: открытки открываются только по
   прямому URL.
5. Сделать commit и push, затем отправить адрес нужной открытки получателю.

Используйте имена директорий в нижнем регистре и относительные пути к общим
ресурсам, чтобы проект корректно открывался как GitHub Project Page.

## GitHub Pages

Workflow `.github/workflows/pages.yml` публикует содержимое репозитория при
push в ветку `main`.

В GitHub откройте **Settings → Pages** и в разделе **Build and deployment**
выберите **Source: GitHub Actions**. После следующего push сайт будет
опубликован автоматически.
